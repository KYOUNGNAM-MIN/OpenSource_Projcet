## Soongsili (Soongsil Chat Bot Project)

### Soongsili 란?
- 숭실대학교 학생들에게 다양한 정보를 효율적이고 빠르게 전달하기 위한 카카오톡 챗봇입니다.

### 아이디어 개요
- 숭실대학교의 정보를 주는 공식 어플은 접근성이 낮고, 에러가 잦아 사용이 불편합니다.
- 따라서, 모바일/웹 모두 접근성이 뛰어난 카카오톡 챗봇으로 숭실대학교의 정보를 제공하기로 합니다.
